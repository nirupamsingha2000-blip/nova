# Wireframes — Modules 1–6 (High-level)

This document contains quick wireframes and interaction notes for the initial modules.

## Module 1 — Explain
- Left: Query input + style selector + level
- Middle: Structured explanation (Definition, Analogy, Simulation, Memory trick, PYQ, Mini-quiz)
- Right: Quick links (Related topics, Next steps, Time estimate)

## Module 2 — Visualize
- Canvas area with play/pause, parameter sliders, preset scenes
- Side panel: explain annotations, step controls, export screenshot

## Module 3 — Practice
- Adaptive question feed
- Progress bar, timer, immediate feedback
- Weakness map summarizing common errors

## Module 4 — Solve
- Upload area (image or text)
- OCR preview, recognized math, step-by-step solver
- Edit recognized input before solve

## Module 5 — Revise
- Flashcard queue (spaced repetition)
- Short micro-explanations and quick quizzes
- Schedule recommendations

## Module 6 — Predict
- Performance forecast graphs
- Suggested study plan and milestone ETA
